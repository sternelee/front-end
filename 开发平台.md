# 前端`nodejs`开发平台

## 安装`nodejs`开发环境

```bash
// 安装`npm`淘宝源
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

## 安装`git`工具，安装`ruby`环境和`python`环境

### 使用`sass`

```bash
gem install sass
```
